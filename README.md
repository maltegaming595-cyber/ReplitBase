# ZosfasGen Web + Bot (Render minimal)

This project runs an Express website + Discord bot in the same process.

## Render
- Build: `npm install` (or `npm ci`)
- Start: `npm start`
- Root Directory: empty

## Env vars (required)
DISCORD_TOKEN=
CLIENT_ID=
DISCORD_CLIENT_SECRET=
DISCORD_REDIRECT_URI=
GUILD_ID=
ROLE_PREMIUM=
MONGODB_URI=
WEBSITE_JWT_SECRET=

Optional:
WEBSITE_ADMIN_ROLE_ID=1467344940383338662
COOKIE_SECURE=true
PORT=10000
